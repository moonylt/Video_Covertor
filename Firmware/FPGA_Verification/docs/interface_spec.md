# 接口时序规范

## 1. DDR3 内存接口

### 1.1 电气规格

| 参数 | 最小值 | 典型值 | 最大值 | 单位 |
|------|--------|--------|--------|------|
| 工作电压 (VDD) | 1.425 | 1.5 | 1.575 | V |
| 参考电压 (VREF) | 0.742 | 0.75 | 0.758 | V |
| 输入高电平 (VIH) | VREF+0.1 | - | VDD+0.3 | V |
| 输入低电平 (VIL) | -0.3 | - | VREF-0.1 | V |

### 1.2 时序参数 (800MT/s)

| 参数 | 符号 | 最小值 | 最大值 | 单位 |
|------|------|--------|--------|------|
| 时钟周期 | tCK | 2.5 | - | ns |
| DQS 到 DQ 建立时间 | tDS | 170 | - | ps |
| DQS 到 DQ 保持时间 | tDH | 170 | - | ps |
| DQ/DQS 输出建立时间 | tDQSCK | -450 | 450 | ps |
| DQ/DQS 输出保持时间 | tQH | 180 | - | ps |

### 1.3 初始化序列

```
1. 上电等待 VDD 稳定 (>200us)
2. 提供稳定时钟 (>500us)
3. 执行 ZQ 校准
4. 执行写电平校准 (WRLVL)
5. 执行读均衡 (Read Leveling)
6. 执行 2D 均衡 (2D Training)
7. 校准完成，进入正常工作模式
```

## 3. SPI Flash 接口

### 3.1 电气规格 (W25Q128JV)

| 参数 | 最小值 | 典型值 | 最大值 | 单位 |
|------|--------|--------|--------|------|
| 工作电压 | 2.7 | 3.3 | 3.6 | V |
| 输入高电平 | 0.7×VCC | - | VCC+0.3 | V |
| 输入低电平 | -0.3 | - | 0.2×VCC | V |

### 3.2 SPI 时序 (标准模式)

| 参数 | 符号 | 最小值 | 最大值 | 单位 |
|------|------|--------|--------|------|
| 时钟频率 | fCLK | - | 50 | MHz |
| 时钟高电平时间 | tCH | 8 | - | ns |
| 时钟低电平时间 | tCL | 8 | - | ns |
| 数据建立时间 | tSU | 2 | - | ns |
| 数据保持时间 | tH | 4 | - | ns |
| 片选建立时间 | tCSH | 20 | - | ns |
| 片选保持时间 | tCSL | 20 | - | ns |

### 3.3 读数据命令序列

```
CS# 拉低:
  [8 位命令：0x03/0x0B]
  [24 位地址：A23-A0]
  [N 位数据：D7-D0, ...]
CS# 拉高
```

### 3.4 写数据命令序列

```
1. 发送写使能 (0x06)
2. CS# 拉高
3. CS# 拉低:
   [8 位命令：0x02]
   [24 位地址：A23-A0]
   [N 位数据：D7-D0, ...]
4. CS# 拉高 (触发内部写入)
5. 等待写入完成 (典型 5ms)
```

## 4. UART 接口

### 4.1 电气规格

| 参数 | 最小值 | 典型值 | 最大值 | 单位 |
|------|--------|--------|--------|------|
| 逻辑高电平 | 2.0 | 3.3 | 3.6 | V |
| 逻辑低电平 | 0 | 0 | 0.8 | V |

### 4.2 波特率配置

系统时钟：50MHz

| 波特率 | 分频系数 | 误差 |
|--------|----------|------|
| 9600 | 5208 | <0.01% |
| 19200 | 2604 | <0.01% |
| 38400 | 1302 | <0.01% |
| 57600 | 868 | <0.01% |
| 115200 | 434 | <0.01% |

### 4.3 数据格式

```
空闲 (高) | 起始位 (0) | D0 | D1 | D2 | D3 | D4 | D5 | D6 | D7 | 停止位 (1) | 空闲 (高)
```

## 7. 约束文件示例

### 7.1 UCF 约束片段

```ucf
# DDR3 约束
NET "ddr3_dq[0]"      LOC = N3   | IOSTANDARD = MOBILE_DDR | TERM = NONE;
NET "ddr3_dqs_p[0]"   LOC = P5   | IOSTANDARD = MOBILE_DDR | TERM = NONE;
NET "ddr3_dqs_n[0]"   LOC = N5   | IOSTANDARD = MOBILE_DDR | TERM = NONE;

# TMDS 约束
NET "tmds_rx_clk_p[0]" LOC = A5  | IOSTANDARD = LVDS_25;
NET "tmds_rx_clk_n[0]" LOC = A6  | IOSTANDARD = LVDS_25;

# SPI Flash 约束
NET "flash_cs_n"      LOC = J14  | IOSTANDARD = LVCMOS33 | SLEW = FAST | DRIVE = 8;
NET "flash_clk"       LOC = K14  | IOSTANDARD = LVCMOS33 | SLEW = FAST | DRIVE = 8;

# UART 约束
NET "uart_tx"         LOC = R14  | IOSTANDARD = LVCMOS33 | SLEW = FAST | DRIVE = 8;
NET "uart_rx"         LOC = T14  | IOSTANDARD = LVCMOS33 | PULLUP;
```

### 7.2 时序约束片段

```ucf
# 时钟约束
TIMESPEC "TS_SYS_CLK" = PERIOD "sys_clk" 10.000 ns HIGH 50%;
TIMESPEC "TS_PIXEL_CLK" = PERIOD "pixel_clk" 13.468 ns HIGH 50%;
TIMESPEC "TS_DDR3" = PERIOD "DDR3_CLK" 2.500 ns HIGH 50%;

# 输入延迟
NET "tfp401_*" TNM_NET = "GRP_TFP401_IN";
TIMESPEC "TS_IN_TFP401" = FROM "GRP_TFP401_IN" TO FFS 2.0 ns;

# 输出延迟
NET "tfp410_*" TNM_NET = "GRP_TFP410_OUT";
TIMESPEC "TS_OUT_TFP410" = FROM FFS TO "GRP_TFP410_OUT" 2.0 ns;
```
