# HDMI 硬件诊断指南

## 诊断目的
快速判断 TFP410 HDMI 输出模块是否正常工作，定位问题来源。

---

## 一、诊断步骤

### 步骤 1: 检查 PLL 锁定状态
**观察 LED[3]**
- ✅ **LED[3] 亮** = PLL 已锁定，时钟正常
- ❌ **LED[3] 灭** = PLL 未锁定，检查晶振和 PLL 配置

### 步骤 2: 检查 HDMI 数据生成
**观察 LED[4]**
- ✅ **LED[4] 亮/闪烁** = HDMI 数据正在输出
- ❌ **LED[4] 灭** = 无数据输出，检查测试图案发生器

### 步骤 3: 切换测试图案
**按 BTN[0] 按键**，观察 LED[0:2] 和显示器变化：

| 模式 | LED[2:0] | 测试图案 | 预期显示器 |
|------|----------|----------|------------|
| 0    | 000      | 全白     | 白色屏幕   |
| 1    | 001      | 全红     | 红色屏幕   |
| 2    | 010      | 全绿     | 绿色屏幕   |
| 3    | 011      | 全蓝     | 蓝色屏幕   |
| 4    | 100      | 8 色彩条  | 彩色竖条纹 |
| 5    | 101      | 网格     | 白色网格   |

### 步骤 4: 测量关键信号
使用示波器/逻辑分析仪测量：

| 测试点 | 信号 | 预期值 | 万用表/示波器测量 |
|--------|------|--------|------------------|
| TP0    | PLL_LOCK | 高电平 (3.3V) | DC 电压 |
| TP1    | PCLK | 74.25MHz 方波 | 频率计/示波器 |
| TP2    | ACTIVE | 脉冲信号 | 示波器 |
| TP3    | DE | 脉冲信号 | 示波器 |

## 三、TFP410 硬件检查清单

### 1. 电源检查 (万用表)
| 引脚 | 电压 | 正常范围 |
|------|------|----------|
| DVDD | 1.8V | 1.71-1.89V |
| DVDDIO | 3.3V | 3.0-3.6V |
| PVDD | 1.8V | 1.71-1.89V |

### 2. 复位检查
- TFP410 RESET 引脚应为高电平 (工作态)

### 3. 时钟检查
- PCLK 输入应有 74.25MHz 信号 (示波器)

### 4. 数据活动检查
- DE/HS/VS 应有活动信号 (示波器)
- D[0:23] 应有数据翻转 (逻辑分析仪)

## 四、常见问题及解决

### 问题 1: LED[3] 不亮 (PLL 未锁定)
**可能原因:**
- 50MHz 晶振未工作
- PLL 配置错误
- 复位未释放

**解决:**
- 测量晶振输出
- 检查 clk_wiz_50to100 配置
- 检查 rst_n 引脚电平

### 问题 2: LED[4] 不亮 (无数据输出)
**可能原因:**
- 复位未释放
- 测试图案发生器未工作
- 像素时钟未到达

**解决:**
- 检查 sys_rst_n 信号
- 检查 pixel_clk_buf 信号
- 重新综合代码

### 问题 3: 显示器无信号
**可能原因:**
- TFP410 电源异常
- HDMI 连接器虚焊
- 电平标准不匹配 (应为 LVCMOS25)
- 显示器不支持 720p@60Hz

**解决:**
- 测量 TFP410 电源引脚
- 检查 HDMI 连接器焊接
- 确认 UCF 中 IOSTANDARD = LVCMOS25
- 尝试其他显示器

### 问题 4: 显示器有信号但无图像
**可能原因:**
- 数据极性错误
- 时序参数不匹配
- RGB 数据位宽问题

**解决:**
- 用示波器检查数据总线活动
- 尝试全白/全红等纯色测试
- 检查 TFP410 数据时序

## 五、快速测试命令

### 1. 查看 LED 状态
```
LED[7:0] = 8'b000LDPA0
              │││││└─ 保留
              ││││└── 测试模式 Bit0
              │││└─── 测试模式 Bit1  
              ││└──── 测试模式 Bit2
              │└───── PLL 锁定 (1=正常)
              └────── HDMI 输出 (1=正常)
```

### 2. 示波器测量点
```
TP0 (AF10): PLL_LOCK  → DC 高电平
TP1 (AG10): PCLK      → 74.25MHz 方波
TP2 (AH10): ACTIVE    → 行脉冲
TP3 (A11):  DE        → 数据使能脉冲
```

### 3. 按键操作
```
BTN[0]: 切换测试图案 (模式 0-5 循环)
BTN[1]: 未使用
RST:    系统复位
```

## 六、正常工作的标志

✅ **所有条件满足表示系统正常:**
1. LED[3] 常亮 (PLL 锁定)
2. LED[4] 常亮/闪烁 (数据输出)
3. 按 BTN[0] 时 LED[0:2] 变化
4. 显示器显示对应测试图案
5. 示波器测得 74.25MHz PCLK 信号

❌ **任一条件不满足需要排查:**
- 检查电源
- 检查时钟
- 检查复位
- 检查约束文件
- 检查 TFP410 硬件连接
