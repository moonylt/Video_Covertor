# ADV7125 VGA 彩条测试项目

## 项目概述
独立的 ADV7125 VGA 输出测试工程，用于验证 VGA 视频 DAC 是否正常工作。

## 功能特性
- **输出信号**: 8 色彩条测试图案
- **彩条顺序**: 白、黄、青、绿、紫、红、蓝、黑 (从左到右)
- **分辨率**: 800x600 @ 60Hz
- **像素时钟**: 40MHz
- **接口**: ADV7125 RGB DAC (**8 位**, 根据原理图)

## 快速开始

### 方法 1: 使用 ISE Project Navigator
1. 打开 Xilinx ISE 14.7
2. 创建新项目：
   - 项目位置：`F:\Video_Covertor\Firmware\FPGA_Verification\projects\adv7125_colorbar`
   - 项目名：`adv7125_colorbar`
   - 顶层模块类型：`HDL`
   - 器件：`XC6SLX45-3FGG484`
3. 添加源文件：
   - `src/adv7125_colorbar_top.v`
   - `src/vga_colorbar_800x600.v`
   - `src/clk_wiz_50to40.v`
   - `src/adv7125_driver_8bit.v`
4. 添加约束文件：
   - `constraints/adv7125_colorbar.ucf`
5. 生成比特流
6. 下载到 FPGA

## 预期结果

### 显示器输出
- 应显示 8 条垂直彩色条纹
- 从左到右：白、黄、青、绿、紫、红、蓝、黑
- 每条彩条宽度约 100 像素 (800/8)

### 示波器测量点
| 测试点 | 信号 | 预期值 |
|--------|------|--------|
| ADV7125_CLK | DAC 时钟 | 40MHz 方波 |
| ADV7125_BLANK_N | 消隐 | 高电平 (有效区) |
| VGA_HS | 行同步 | 37.88kHz 脉冲 |
| VGA_VS | 场同步 | 60.32Hz 脉冲 |
| ADV7125_RED[7] | 数据 | ~20MHz 活动 |

## 故障排查

### 问题 1: 显示器无信号
**检查项:**
1. 测量 ADV7125_CLK 是否有 40MHz 信号
2. 检查 VGA_HS/VGA_VS 是否有同步脉冲
3. 确认显示器输入源选择正确 (VGA/Analog)
4. 检查 VGA 连接器是否插好

### 问题 2: 有信号但无图像
**检查项:**
1. 用示波器测量 RGB 数据线是否有活动
2. 检查 ADV7125 电源电压 (3.3V 模拟/数字)
3. 确认 ADV7125_RESET 引脚为高电平
4. 检查 ADV7125 参考电流设置电阻 (R87 = 1kΩ)

### 问题 3: 图像颜色不正确
**检查项:**
1. 确认 RGB 数据线序正确 (8 位)
2. 检查 PCB 焊接是否有短路/开路
3. 验证 ADV7125 的 BLANK_N 信号极性
4. 检查电阻排 RN14-RN20 是否焊接正确

## ADV7125 关键引脚

| 引脚 | 功能 | 电压 | 说明 |
|------|------|------|------|
| DVDD (13,25,28,31,34,39,40) | 数字电源 | 3.3V | 数字部分供电 |
| AVDD (14,26,27,32,33,38) | 模拟电源 | 3.3V | 模拟部分供电 |
| RESET (35) | 复位 | 3.3V | 高电平工作 |
| CLK (24) | 时钟输入 | 3.3V | 40MHz 像素时钟 |
| BLANK_N (11) | 消隐 | 3.3V | 低电平关闭输出 |
| R[7:0] (48:41) | 红色数据 | 3.3V | 8 位 RGB 数据 |
| G[7:0] (10:3) | 绿色数据 | 3.3V | 8 位 RGB 数据 |
| B[7:0] (23:16) | 蓝色数据 | 3.3V | 8 位 RGB 数据 |
| VREF (36) | 参考电压 | 1.24V | 内部带隙基准 |
